Console

Console 面板主要的功能可以显示 Log 信息, 运行 JavaScript, Console 面板可以以专有模式单独显示或者以抽屉模式和其他面板一起使用。
Log Messages

控制台中除了使用 Console API 显式输出的 Log Message, 也包含了浏览器在执行页面代码时遇到的network error,未捕获的异常以及其他信息。

可以对Console中的Message进行多种形式的过滤 —— 来源, 级别, 正则 etc.

可以自动折叠Message, 或者对每条Message显示TimeStamp, 可以勾选 Preserve Log来保留页面跳转、刷新前后的 Log

Run JavaScript

Console 是一个 REPL (Read–eval–print loop), also termed an interactive toplevel or language shell 它读取您键入的JavaScript，评估代码，打印出表达式的结果，然后循环回到第一步。

Console 可以访问对应 Page 的 Window 对象
