Network

通过此面板查看和调试网络活动

Network DEMO
基本功能
  • 记录并查看网络活动, 按条件过滤网络活动
  • 记录并查看网络活动的明细信息
  • Block 特定的网络请求或者特定Domain的网络请求
  • 网络速度模拟/截图/强制缓存失效/保留跨页面请求记录
  • 可以对XHR类型的请求进行重放
  • 将请求/响应数据复制到剪贴板
  • 查看请求来源
  • 查看加载事件,查看请求数,查看总下载大小(压缩后),查看请求的未压缩大小
  • 进行高级过滤：method, status-code, larger-than, domain 等等
  • 常见网络问题排查
单个请求的不同阶段
  • Queueing 排队阶段, 排队原因： 优先级, http1.0/1.1 TCP连接<=6, 浏览器正在磁盘缓存中短暂分配空间
  • Stalled 停滞阶段：该请求可能由于“ 排队”中描述的任何原因而停止。
  • (Optional) 代理协商： 它表示代理服务器连接协商所用的时间
  • DNS查找：解析请求对应的IP地址
  • 初始化连接： TCP握手, SSL协商
  • Request Sent: 网络进程准备请求数据，并进行发送。这个阶段通常很快。
  • Waiting(TTFB/Time to first byte) 此时间包括一次往返延迟和服务器准备响应所花费的时间。
  • Content Download
